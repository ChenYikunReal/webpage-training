# 简单网页开发训练

- IDE：WebStorm
- 内容：基本的Web前端开发训练

## 目录结构说明

- basic_html_tags : HTML基本标签使用
- brython_test : Brython库的使用
- center_display : 网页全屏和居中功能
- crazy_code : 《疯狂HTML5+CSS3+JavaScript讲义》配书代码
- css_lib : CSS库积累
- css_training : CSS使用训练
- font_awesome : 图标字体库和CSS框架FontAwesome使用
- google_map : 地图接口使用
- hello_html : HTML入门代码
- info_download : 一些不易访问的网页下载
- js_lib : JavaScript库积累
- js_training : JavaScript使用训练
- material : 某些网页特效
- multimedia : 多媒体网页相关
- web_template : 网页模板积累
